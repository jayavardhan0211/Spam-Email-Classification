# 📨 Spam Email Classification

A lightweight Machine Learning project that classifies emails as **Spam** or **Ham (Legitimate)** using Natural Language Processing (NLP) and supervised learning.

## 🚀 Features
* **Text Preprocessing:** Tokenization, stop-word removal, and TF-IDF vectorization.
* **High Accuracy:** Built using Scikit-Learn (Naive Bayes / Logistic Regression).
* **Quick Inference:** Easily test custom email strings to check if they are spam.

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Scikit-Learn, Pandas, NumPy, NLTK
